# Project Overview

## Summary

SkyGrid-UAV-Simulator is a research-oriented simulation project built on OMNeT++ and the INET framework. It is designed to model cooperative UAV systems, wireless communication behavior, mobile sensors, and protocol-driven data collection workflows.

## Main Use Cases

- Testing UAV coordination strategies before real deployment
- Comparing communication and mobility protocol behavior
- Studying sensor data collection efficiency
- Running controlled simulation campaigns for performance analysis

## Core Components

- `src/gradys_simulations/`: main simulation modules and protocol logic
- `showcases/`: ready-to-run example scenarios
- `documentation/`: diagrams, media, and documentation pages

## Why This Repository Matters

This project helps bridge networking simulation with more realistic UAV movement behavior. That makes it useful for academic research, experimental protocol design, and early validation of swarm-based sensing systems.
